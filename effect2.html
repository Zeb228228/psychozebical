Web Design Flashcards

What is HTML?
HyperText Markup Language used to structure web content.
What does CSS do?
It styles HTML elements (colors, layout, fonts, etc.).
What is JavaScript?
A programming language for interactivity in web pages.
What is a div?
A container element used to group content in HTML.
What is a class in CSS?
A reusable style group applied to HTML elements using "class" attribute.
What is responsive design?
Design that adjusts for different screen sizes/devices.
What is an ID in HTML?
A unique identifier for one element, referenced with # in CSS/JS.
What is a hyperlink?
A link to another page or website, made with the <a> tag.
